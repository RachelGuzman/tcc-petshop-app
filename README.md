# TCC - Aplicativo de Petshop

Projeto desenvolvido para a disciplina de Trabalho de Conclusão do Curso Software Product: Analysis, Specification, Project & Implementation.

## Objetivo
Desenvolver um aplicativo simples para gerenciamento de um petshop. 

## Tecnologias Utilizadas
- Back-end: Java
- Front-end: React
- Banco de Dados: MySQL
  - petshop.sql -> Executar no MySQL para criar todas as tabelas do projeto.

## Funcionalidades
1. Cadastro de Clientes
2. Cadastro de Pets
3. Agendamento de Serviços
4. Cancelamento de Agendamento
